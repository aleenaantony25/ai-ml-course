# Supervised Machine Learning

This folder contains supervised learning algorithms implemented using Python and scikit-learn.

## Topics
- Linear Regression
- Feature Engineering
- Model Training & Evaluation
- Regularization

## Files
- `linear_regression.ipynb` – End-to-end implementation of linear regression with feature engineering
- `house_price_prediction.ipynb` – House price prediction project using Linear Regression, including data preprocessing, handling missing values, feature encoding, model training, and evaluation
- `lasso_regression_insurance.ipynb` – Lasso Regression model to predict insurance charges with feature encoding, interaction terms, regularization tuning using multiple alpha values, and cross-validation (LassoCV)
